# High Performance Computing for Data Science

This repository contains exercises and tasks from the HPC course on MPI and OpenMp at the University of Trento.

Furthermore, it contains the course project:

## Parallel Hough Transform

This project is for the HPC course at the University of Trento (2025 fall).

The algorithm is parallelized independently using **MPI** and **OpenMP**.

- hough_transform.c - sequential code
- canny - preprocessing
- libs - stb_image.h
- mpi - mpi implementation and queueing scripts
- omp - omp implementation and queueing scripts
- \*\_out\_\* - output of runs
- calculations - processing scripts and results
